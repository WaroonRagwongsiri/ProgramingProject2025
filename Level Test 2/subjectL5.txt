L5 : ให้นักศึกษาเขียนโปรแกรมรับ string 2 ชุด (string , pattern) โดย string มีเฉพาะตัวเลขและ
ตัวอักษร และ pattern ใช้เฉพาะตัวอักษร d และ c เท่านั้น โดยตัว d หมายถึงตัวเลข และ c หมายถึง
ตัวอักษร แล้วโปรแกรมตรวจสอบว่าใน string มีลักษณะของข้อมูลที่ตรงตาม pattern คือค่าใด
(แสดงผลเฉพาะค่าแรก ถ้าไม่มีให้แสดงผลว่า None)
Input :
String = abcd1234abc80cdef1
Pattern = cccddcc
Output : abc80cd
